# StreamUI - Vanilla JS Video Platform

## Features
- SPA Router (Vanilla JS)
- Infinite Scroll Feed
- Debounced Search
- Authentication (Mock JWT)
- Protected Routes
- Role-based UI (Admin/User)
- Lazy Loading Images

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- IntersectionObserver API

## Architecture
Custom SPA architecture with:
- Router system
- Page lifecycle
- Component-based UI
